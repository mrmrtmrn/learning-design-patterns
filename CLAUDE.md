# CLAUDE.md

## リポジトリ概要

GoFのデザインパターンを、書籍「Java言語で学ぶデザインパターン入門 第3版」を参考にしながらJava・Rubyで実装して学ぶリポジトリ。

## ディレクトリ構成

- `java/` - Javaの実装コード（パターンごとにサブディレクトリ）
- `ruby/` - Rubyの実装コード（パターンごとにサブディレクトリ）

## 実行環境

- Docker を使用（Java / Ruby それぞれのコンテナ）
- ローカルのソースコードをボリュームマウントしてコンテナ内で実行する構成
- プロジェクトルートで `./run` スクリプトを使って実行する

```bash
./run java/iterator/Main.java
./run ruby/iterator/main.rb
```

## Git ワークフロー

`main` ブランチで直接運用。ブランチ運用やPRは不要。
